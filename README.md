# Biodiversity-Species-Prediction
Machine learning project predicting species presence using TerraClimate data and soil variables. Developed with Python, Scikit-learn, and Pandas

## 🎯 Project Objective
This project focuses on predicting species presence by analyzing complex environmental datasets. The goal was to build a robust machine learning model to support biodiversity conservation efforts by identifying key environmental drivers.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
* **Data Sources:** TerraClimate data, Soil variables (pH, texture, moisture, carbon content)

## 🔍 Methodology
1. **Data Cleaning & Preprocessing:** Handled missing values and performed feature scaling to ensure data integrity.
2. **Feature Engineering:** Created derived environmental variables based on soil and climate interactions to enhance model predictive power.
3. **Modeling:** Implemented supervised learning algorithms (Regression/Classification) to predict species occurrence.
4. **Evaluation:** Validated model performance using cross-validation techniques and optimized hyperparameters for accuracy.

## 📈 Key Insights
* Identified critical soil and climate variables that significantly impact species distribution.
* The model demonstrates strong potential for assisting ecological monitoring and conservation planning.
